# ContactVault

A full-stack contact management system built with Java and 
Spring Boot — featuring secure OAuth2 login, cloud image 
storage, and Excel export functionality.

## Features
-  OAuth2 Authentication (Google Login)
- Add, Edit, Delete & Search Contacts
- Cloud image upload via Cloudinary API
- Export contacts to Excel (.xlsx)
- Responsive UI with Tailwind CSS

## Tech Stack
| Layer      | Technology              |
|------------|-------------------------|
| Backend    | Java, Spring Boot       |
| Frontend   | Thymeleaf, Tailwind CSS |
| Database   | MySQL, JPA/Hibernate    |
| Auth       | Spring Security, OAuth2 |
| Cloud      | Cloudinary API          |
| Tools      | Git, Maven              |
